in c the size_t : "size type" represnt the size of an object in bytes is an unsigned integer type of at least 16 bit 

in lemons terms this sentence mean : 
fread(...) means read some data from a file.

buffer → where we put the data we read.

1 → each “thing” we read is 1 byte big.

BLOCK_SIZE → how many of those 1-byte things we want to read (so total = BLOCK_SIZE bytes).

fptr → the file we’re reading from.

> 0 → means “did we actually read something?” (if yes, continue).

while ((bytes_read = fread(buffer, 1, BLOCK_SIZE, fptr)) > 0) {
